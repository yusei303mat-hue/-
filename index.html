システム設定

安全な動作のために、ご自身の Gemini APIキー をご設定ください。キーはあなたのブラウザのみに保存され、外部のサーバーには送信されません。

※ キーがない場合は、Google AI Studio で無料で即座に取得できます。

スマホや別のパソコンとデータを安全に同期します。データはあなた自身のGoogle Driveに hiyori_backup.json として非公開保存されるため、完全に安全です。

クラウド同期状況 未連携
設定手順：
1. Google Cloud Console でプロジェクトを作成
2. Google Drive API を有効にする
3. OAuth 同意画面を設定し、認証情報の作成から「OAuth クライアント ID」(ウェブ アプリケーション) を取得
4. 「承認済みの JavaScript 生成元」に、あなたがこのHTMLを公開するURL (GitHub Pagesなど。ローカル検証なら http://localhost や http://127.0.0.1) を登録。
